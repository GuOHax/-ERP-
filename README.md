项目内容：
基于 Spring Boot+Vue 3+Element Plus 构建了一套完整的企业资源管理平台，实现员工信息管理、部门架构管理、角色权限管理、销售/采购/库存业务管理等功能模块，覆盖企业日常运营核心场景。
工作内容：
后端架构:基于Spring Boot 3+JPA+ MySQL设计并实现 RESTful API, 包含 Employee, Department, Role, Sales, Purchase, Inventory等核心业务实体，覆盖完整的CRUD操作及业务逻辑处理。
前端开发:使用 Vue 3 Composition API+Element Plus 构建响应式管理界面，实现列表分页搜索、表单弹窗编辑、头像上传、状态切换等交互功能。
·权限与安全:配置Spring Security实现接口级别的访问控制，支持CORS跨域与无状态会话。
数据管理:通过JPA自动建表与字段扩展，前端下拉联动实时加载部门数据，实现员工与部门的关联绑定。
项目成果:
完成系统管理、销售管理、采购管理、库存管理5大模块的完整开发与页面美化。
实现了员工、部门、角色等数据的增删改查，以及采购审核流程、库存补货等业务闭环。
项目结构清晰，模块解耦，可扩展性强，已完整运行并具备实际使用价值。
